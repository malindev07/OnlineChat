# Онлайн чат
Онлайн чат для общения 2-х или более пользователей.  
Для запуска установить заивисимости, запустить серверную часть: uvicorn main:app, заупстить gui


##### Использованные технологии:

1. Backend
	1. python
	2. fastapi
	3. multithreading
	4. http
	5. pydantic
    
2. GUI
    1. tkinter 
    2. mttkinter 
3. DB
    1. Postgres
    2. sqlalchemy
    3. alembic
